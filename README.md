## About App 

### technology used:
###  nestjs-typescript-typeORM-postgres-nodemailer


---
a big nestjs app where you can:

- login and register with email and password(jwt-Service)
- reset and forget password by sending email(mailtrap)
- use Guard and Interceptor to authorization user by roles and permission
- category and subcategory with parent id
- Cart Module
- Send email for Submitted order (mailtrap)
- order and order detail
- Review of the purchased product with rating and comment
- product and product gallery
- upload file




## Installation
clone this repository <br />
npm install  <br />
create postgres database and connect project to it <br />
import postman collection to postman <br />
npm run start:dev  <br />
